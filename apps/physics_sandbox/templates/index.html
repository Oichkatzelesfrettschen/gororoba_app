Physics Sandbox

Deterministic oscillator lab for visual intuition and benchmarkable outputs. Use this as a stepping stone toward deeper simulation engines.

Learning layer: Explorer Mode (toggle below for Story, Explorer, and Research depth).

Story Layer

Think of a spring toy that bounces and slowly settles down because air and friction steal energy.

Mini activity: predict whether stronger damping will settle faster before you run the simulation.

Explorer Layer

This model computes acceleration as a = -k*x - c*v, then steps velocity and position forward in time.

  1. Run a baseline with default settings.
  2. Increase damping and compare settling speed.
  3. Increase spring stiffness and compare oscillation frequency.

Research Layer

Governing ODE: x'' + c*x' + k*x = 0 with explicit Euler integration under fixed dt.

Assumptions

  • Unit mass with linear spring and linear viscous damping.
  • No external forcing term.
  • Numerical stability depends on selected dt, k, and c.
